Данный справочник предназначен для хранения данных о форматах графических файлов. Кнопка "Открыть справочник" откроет окно для работы со справочником. В этом окне возможно добавление, удаление, изменение, поиск и вывод записей. Так же в окне выводится информация о том, файлов какого формата больше. Для поиска записи, введите в поле расширение искомого формата и нажмите кнопку "поиск записи". Для изменения записи, введите в соответствующее поле названия формата и нажмите кнопку "изменить запись". В открывшемся окне имените нужные поля, и нажмите кнопку "изменить". Для удаления записи, введите в соответствующее поле названия формата и нажмите кнопку "удалить запись". Для просмотра всего списка нажмите кнопку "просмотреть весь список". Для добавления новой записи нажмите кнопу "добавить запись", а в открывшемся окне заполните все поля и нажмите кнопку "добавить". Кнопка "Задание" в главном меню открывает окно с заданием. Кнопка "О программе" выводит в главное меню текущую версию программы и координаты автора. Кнопка "Выход" закрывает программу.
